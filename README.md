# Python Project in Cursor

Welcome to your Python development environment in Cursor!

## Getting Started

### 1. Install Python
If you haven't already, install Python from [python.org](https://www.python.org/downloads/).
Make sure to check "Add Python to PATH" during installation.

### 2. Run Your Code
You can run Python code in several ways:

#### Option A: Using Cursor's Terminal
1. Open the terminal in Cursor (Ctrl + `)
2. Navigate to your project directory
3. Run: `python main.py`

#### Option B: Using Cursor's Run Button
1. Open `main.py`
2. Click the "Run" button (▶️) in the top right corner
3. Or use the keyboard shortcut (usually F5)

#### Option C: Using Python Interactive Mode
1. Open terminal in Cursor
2. Type `python` to enter interactive mode
3. Import and run your functions

### 3. Install Dependencies
When you need additional packages:
1. Add them to `requirements.txt`
2. Run: `python -m pip install -r requirements.txt`
